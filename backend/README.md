# Dataset Generator Backend

## Installing / Getting started
    > git clone https://github.com/dudursn/dataset-generator.git
    > cd dataset-generator/backend
    > pip install -r requirements.txt
    -- Executar sem debug
    > flask run
    -- Executar com debug
    > flask --debug run
